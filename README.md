# Accessibility CMS
A Django-based web accessibility auditing tool with dashboard, pages list, audit history, and setup for automation tools.

## Features
- Dashboard UI
- Pages overview
- Manual audit input form
- Audit history tracking
- Tailwind CSS UI

## Tech Stack
- Django
- Tailwind CSS
- HTML Templates

## Setup Instructions
1. Create virtual env:
```
python -m venv venv
```
2. Install dependencies:
```
pip install django
```
3. Run project:
```
python manage.py runserver
```
